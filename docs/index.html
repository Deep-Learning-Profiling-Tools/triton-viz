Triton-Viz
Open Source | MIT Licensed

Visualize and debug your Triton GPU kernels

Interactive 3D tensor visualization, memory access patterns, and operation analysis for Triton kernel development. No GPU required.

pip install triton-viz

Visualization Features

Interactive 3D rendering allows you to inspect tensor layouts and memory access patterns from any angle.

Visualize tensor values with color gradients to quickly identify outliers, zeros, or saturation.

Analyze the distribution of values within your tensors to understand data characteristics and debugging issues.

Directly map visual operations back to your source code lines for seamless debugging.

Filter and inspect specific Program IDs (PIDs) to understand parallel execution and block-level behavior.

Easy Integration

Add two lines of code to visualize any Triton kernel

trace and launch
import triton_viz
from triton_viz.clients import Tracer

# 1. Decorate your kernel
@triton_viz.trace(client=Tracer())
@triton.jit
def add_kernel(x_ptr, y_ptr, output_ptr, ...):
    ...

# 2. Launch the visualizer
triton_viz.launch()
run an example
# Clone and run the GEMM example
git clone https://github.com/Deep-Learning-Profiling-Tools/triton-viz.git
cd triton-viz
uv run examples/gemm.py

Supported Operations

Comprehensive visualization for all major Triton operations

tl.load
Memory load patterns with mask visualization
tl.store
Memory store patterns and access tracking
tl.dot
Matrix multiplication with A, B, C visualization
tl.flip
Tensor dimension flipping operations
NKI Flow
Memory hierarchy flow for Neuron devices
Reduce Ops
Sum, min, max reduction operations
