Visualize and debug your Triton GPU kernels

Interactive 3D tensor visualization, memory access patterns, and operation analysis for Triton kernel development. No GPU required.

Quick Install
git clone https://github.com/Deep-Learning-Profiling-Tools/triton-viz.git && cd triton-viz && uv sync

Powerful Visualization Features

Everything you need to understand and optimize your Triton kernels

📊

3D Tensor View

Interactive WebGL rendering with real-time hover detection, camera controls, and persistent view states.

🎨

Value Heatmaps

Color tensors by value with per-operation color schemes and multi-program access count visualization.

📈

Histogram Analysis

Analyze value distributions with configurable bins, statistics display, and per-tensor selection.

📐

CAD-style Dimensions

Tensor shapes annotated with extension lines, arrowheads, and color-coded legends for easy identification.

🔍

Memory Analysis

Track operation counts, memory usage, scratch buffer timelines, and out-of-bounds access detection.

💻

Code Context

Source code peek panel with line highlighting for each operation, prioritizing user code over framework internals.

How It Works

From kernel code to visual insights in four simple steps

1

Decorate

Add @triton_viz.trace decorator to your Triton kernel function

2

Execute

Run your kernel with TRITON_INTERPRET=1 to trace operations

3

Launch

Call triton_viz.launch() to start the visualization server

4

Explore

Navigate programs, inspect tensors, and analyze memory patterns

Supported Operations

Comprehensive visualization for all major Triton operations

tl.load
Memory load patterns with mask visualization
tl.store
Memory store patterns and access tracking
tl.dot
Matrix multiplication with A, B, C visualization
tl.flip
Tensor dimension flipping operations
NKI Flow
Memory hierarchy flow for Neuron devices
Reduce Ops
Sum, min, max reduction operations
